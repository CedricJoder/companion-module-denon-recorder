## Denon DN-500R, DN-700R, DN-900R

This module controls a Denon USB/SD recorder.

The DN-500R will require an Ethernet to RS232 adapter.

**Available commands**
* Power On/Off
* Recording
* Playback 
* Panel Lock/Unlock 

**Presets Included:**
* Presets for most commands

**Feedback Included:**
* Playing / Paused
* Recording / Paused
* Power On / Off

**Missing parts:**
* Configuration commands
